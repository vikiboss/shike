# Shíkè 「時刻」

Zero dependency & pure date-time formatting library, nothing redundant.

## Installation

```bash
# pnpm
pnpm add shike
# npm
npm i shike
# yarn
yarn add shike
```

## Usage

```tsx
import shike from 'shike';

console.log(shike(new Date(), 'YYYY-MM-DD HH:mm:ss'));
```
